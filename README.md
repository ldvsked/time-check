# time-check
## README: Помощник времени для программиста

**Описание проекта**
Набор функций-помощников для программиста Арсения, который потерял счет времени.

**Функционал**
- `currentDayOfTheWeek()` — возвращает текущий день недели на русском
- `dayOrNight()` — определяет, день (10:00–22:00) или ночь
- `nextFriday()` — считает дни до следующей пятницы (0, если сегодня)
- `CheckCurrentDayOfTheWeek()` — проверяет угадал ли Арсений день недели
- `CheckNowDayOrNight()` — проверяет угадал ли Арсений время суток (с валидацией длины строки)

**Технологии**
- Язык: Go
- Пакеты: time, strings, unicode/utf8, errors

**Особенности**
- Регистронезависимые проверки
- Обработка ошибок при неверной длине строки
- Учет совиного графика (день с 10 до 22)
